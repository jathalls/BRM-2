### Syncfusion.Maui.Maps Component

This package contains [.NET MAUI Maps](https://www.syncfusion.com/maui-controls/maui-maps) component for .NET MAUI application.

### System Requirements

* [System Requirements](https://help.syncfusion.com/maui/system-requirements)

### Overview

The Syncfusion [.NET MAUI Maps](https://www.syncfusion.com/maui-controls/maui-maps) control contains statistical information for a geographical area for .NET MAUI application.
