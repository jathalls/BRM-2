### Syncfusion.Maui.Sliders Components

The package contains [.NET MAUI Slider](https://www.syncfusion.com/maui-controls/maui-slider), [.NET MAUI Range Slider](https://www.syncfusion.com/maui-controls/maui-range-slider), and [.NET MAUI Range Selector](https://www.syncfusion.com/maui-controls/maui-range-selector) components for the .NET MAUI application.
