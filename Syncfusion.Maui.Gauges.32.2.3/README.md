### Syncfusion.Maui.Gauges Components

This package contains [.NET MAUI Linear Gauge](https://www.syncfusion.com/maui-controls/maui-linear-gauge), [.NET MAUI Radial Gauge](https://www.syncfusion.com/maui-controls/maui-radial-gauge) and [.NET MAUI Digital Gauge](https://www.syncfusion.com/maui-controls/maui-digital-gauge) components for .NET MAUI application.
